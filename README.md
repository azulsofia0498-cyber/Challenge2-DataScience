# Challenge2-DataScience
## Descripción

En el sector de telecomunicaciones, **retener clientes es clave para la rentabilidad**, ya que adquirir nuevos usuarios implica altos costos.  
Este proyecto analiza datos de **clientes de Telecom X** para identificar **factores asociados al abandono de clientes (Churn)**.

A partir de datos demográficos, servicios contratados y facturación, se realiza **limpieza de datos, análisis descriptivo y visualización** para detectar patrones que permitan comprender el comportamiento de los clientes y generar **insights útiles para la toma de decisiones**.

## Relevancia del proyecto

- Diseñar **estrategias de fidelización más efectivas**.
- Identificar **segmentos de clientes con mayor riesgo de abandono**.
- **Optimizar servicios y planes** según el comportamiento de los usuarios.
- Apoyar **decisiones comerciales basadas en datos**.
